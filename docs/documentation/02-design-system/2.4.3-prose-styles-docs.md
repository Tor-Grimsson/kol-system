# Documentation Prose Spec

**Purpose:** Wiki-style typography system extracted from `/apps/web/src/routes/foundry/prose-specs/DocumentationProseSpecs.jsx`. Use this when implementing docs navigation, article content, or any structured knowledge base surfaces.

> The measurements below map directly to the `.doc-*` classes defined in `packages/ui/css/docs.css`. Apply them on top of the Stack prose wrappers when you need deterministic typography outside of CMS-driven content.

## Usage Notes

- Combine `.kol-prose` (65ch wrapper) with `.doc-*` classes for predictable long-form documentation layouts.
- Ordered lists use decimal-leading-zero markers (`01`, `02`, `03`) via the custom docs styles; unordered lists retain the bold plus marker.
- The `.docs-codeblock-wrapper` component supplies scroll affordances and the subtle background defined in `packages/ui/css/docs.css`.
- Apply `.text-fg-64` or `.text-fg-48` utilities for supporting text (captions, metadata) instead of hardcoding opacity values.
- Reference `/apps/web/src/routes/foundry/prose-specs/DocumentationProseSpecs.jsx` if you need the rendered example markup used across Foundry.
