# Málrómur Editorial System

**Purpose:** Document the serif-based typography system showcased in `/apps/web/src/routes/foundry/prose-specs/MalromurProseSpecs.jsx`. Use this file when implementing print-inspired editorial layouts, newsletters, or scholarly spreads that rely on the TG Málrómur font.

> All measurements sit on the shared 8pt baseline grid. Sizes below list both pixel and rem equivalents plus the resulting `em` line height so you can recreate the exact feel outside the Foundry specimen.

## Usage Notes

- Apply `.kol-prose-wide` when recreating the specimen layouts—the 90ch wrapper best matches the editorial spreads presented in Foundry.
- Keep list spacing consistent with the 8pt grid; nested lists inherit the same `1rem` indent used in the Stack system.
- Blockquotes use the serif font with a subtle `border-left` accent (2px standard) and pair with mono attribution text when needed.
- The `.editorial-sidebar-label` uses positive tracking (`50` = `0.05em`) to mimic the print-inspired uppercase tabs—always render it in uppercase.
- Reference the Foundry specimen when designing interactive demos: it showcases how epigraphs, pull quotes, and captions combine across multi-column layouts.
