## Usage Notes

- All helper classes are fixed-size (no `clamp()`), making them ideal for badges, pills, and UI controls.
- Display/heading/text classes inherit semantic colors, so pair them with `.text-auto` inside inverse contexts instead of overriding color tokens manually.
- Deprecated aliases (`.kol-text`, `.kol-body`, `.kol-label`, etc.) remain documented in [2.2.0](2.2.0-typography.md#legacy-aliases-deprecated); avoid them in new work.
- When in doubt about width or rhythm, combine these classes with the prose wrappers described in `2.4.2+` for predictable line lengths.
