**Version:** 1.0.0
**Date:** 2025-12-02
**Status:** Active
**Content Type:** page-documentation

## Overview

Documentation for the `/workshop/design-system/documentation` page - the internal documentation viewer that renders markdown files from `docs/documentation/`.

## Routes

- **Index:** `/workshop/design-system/documentation`
- **Reader:** `/workshop/design-system/documentation/:docId`

## Architecture

### Layout System

Three-column responsive layout:

| Column | Width | Purpose |
|--------|-------|---------|
| Nav (left) | 304px | Collapsible navigation by category |
| Main | flexible | Article content |
| TOC (right) | 192px | "On this page" section links |

Mobile breakpoints collapse columns into drawer components.

### Key Components

**Page Components** (`apps/web/src/routes/workshop/`):
- `Documentations.jsx` - Index page listing all docs
- `DocumentationReader.jsx` - Individual doc viewer

**Layout Components** (`apps/web/src/components/workshop/docs/`):
- `DocsLayout.jsx` - Three-column grid system
- `DocsPageHeader.jsx` - Header with curated tabs
- `DocsToc.jsx` - Table of contents with IntersectionObserver
- `DocsArticle.jsx` - Content wrapper
- `DocsRailDrawer.jsx` - Mobile navigation drawers

### Data Layer

**documentationInventory.js** (`apps/web/src/data/workshop/`):
- Uses Vite glob import: `@docs/documentation/0[0-7]-*/**/*.md`
- Parses frontmatter metadata from markdown
- Generates unique IDs (folder-prefixed for index.md files)

**docsTabBuilder.js** (`apps/web/src/utils/`):
- Curated highlight tabs for essential docs
- Maps doc IDs to navigation links

### Markdown Parser

Custom parser (`parseDocsMarkdown.jsx`) supports:
- H1-H4 headings with ID generation
- Code blocks with syntax highlighting
- Lists (ordered/unordered)
- Blockquotes and callouts
- Tables
- Inline formatting (bold, italic, code, links)

## Folder Structure

Documentation lives in `docs/documentation/` with numbered folders:

```
docs/documentation/
├── 00-metadata/        # Documentation system meta
├── 01-foundation/      # Architecture & setup
├── 02-design-system/   # Visual language
├── 03-components/      # UI components
├── 04-pages/           # Public site pages
│   ├── foundry/        # Type foundry
│   └── collections/    # Collections
├── 05-workshop/        # Workshop (this section)
├── 06-research/        # Research docs
├── 07-operations/      # Development & ops
└── archive/            # Archived docs (excluded from viewer)
```
