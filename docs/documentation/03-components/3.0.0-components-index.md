# 3.0.0 Components: Components Overview

A library of reusable UI components built on atomic design principles.

---

## Structure

Components are organized into three levels:

- [Atoms](/workshop/components/atoms) — Basic building blocks (buttons, inputs, icons)
- [Molecules](/workshop/components/molecules) — Composite components (cards, control panels)
- [Organisms](/workshop/components/organisms) — Complex features (grids, carousels, filters)

---

## Import

```jsx
import { Button, Icon, Divider } from '@kol/ui'
```
